## Prerequisites

- **Node.js 18+** — [nodejs.org](https://nodejs.org)
- **npm** — comes with Node.js
- **OpenRouter account** — free, takes 60 seconds

Check you have Node:
```bash
node --version   # Should show v18.x.x or higher
npm --version
```

---

## Step 1 — Get Your Free OpenRouter API Key

1. Go to **[https://openrouter.ai](https://openrouter.ai)**
2. Click **Sign In** → create a free account
3. Go to **[https://openrouter.ai/keys](https://openrouter.ai/keys)**
4. Click **Create Key**, name it `ANORA`, copy the key
5. The key looks like: `sk-or-v1-abc123...`

> **Free Tier Note:** OpenRouter's free tier includes Claude Sonnet 4.6
> with generous daily limits — more than enough for personal use.

---

## Step 2 — Project Setup

### Option A: Fresh install (recommended)

```bash
# Create project folder
mkdir anora-agent && cd anora-agent

# Copy in the three files you received:
#   anora.js
#   package.json
#   .env.example

# Install dependencies
npm install
```

### Option B: Clone from a repo

```bash
git clone <your-repo-url> anora-agent
cd anora-agent
npm install
```

---

## Step 3 — Configure Your API Key

Create a `.env` file in the project root:

```bash
# On Mac/Linux:
cp .env.example .env

# On Windows (PowerShell):
copy .env.example .env
```

Open `.env` and replace the placeholder:

```env
OPENROUTER_API_KEY=sk-or-v1-your-actual-key-here
```

> ⚠️ **Security:** Never share your `.env` file. Add it to `.gitignore`:
> ```bash
> echo ".env" >> .gitignore
> ```

---

## Step 4 — Run ANORA

```bash
npm start
```

You should see the ANORA banner and a warm greeting. You're live! 🎉

## Troubleshooting

### "OPENROUTER_API_KEY is not set"
- Make sure `.env` exists in the same folder as `anora.js`
- Make sure the key starts with `sk-or-v1-`
- No spaces around the `=` sign

### "Authentication failed (401)"
- Your API key is wrong or expired
- Go to [openrouter.ai/keys](https://openrouter.ai/keys) and create a new one

### "Rate limit hit (429)"
- You've hit the free tier limit — wait a few minutes
- Or upgrade at [openrouter.ai](https://openrouter.ai) for higher limits

### "Cannot find package 'ai'"
```bash
# Delete node_modules and reinstall
rm -rf node_modules
npm install
```

### "SyntaxError: Cannot use import statement"
- Make sure you're on **Node.js 18+**
- Make sure `"type": "module"` is in your `package.json` (it is by default)

## Customizing ANORA

### Change the model
In `anora.js`, find:
```js
const MODEL = "anthropic/claude-sonnet-4-5";
```
Replace with any model from [openrouter.ai/models](https://openrouter.ai/models).

### Change ANORA's personality
Edit the `SYSTEM_PROMPT` constant in `anora.js`.

### Add new commands
In the `switch (cmd.toLowerCase())` block, add new `case` entries.

## Deployment Options

### Local (default)
```bash
npm start
```

### Keep running in background (Linux/Mac)
```bash
npm install -g pm2
pm2 start anora.js --name anora
pm2 logs anora
```

### Docker
```dockerfile
FROM node:20-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
CMD ["node", "anora.js"]
```

```bash
docker build -t anora .
docker run -it --env-file .env anora
```
